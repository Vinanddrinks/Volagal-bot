# Volagal bot
 volagal discord bot

# how to config

the bot expect to find a config.json file answering to this syntax at the root of the package :

{"BOT_PROPERTIES":{
    "TOKEN": "Your Token Here",
    "CONSOLE_INFO":"VOLABOT Info >",
    "CONSOLE_WARN":"VOLABOT Warn >",
    "CONSOLE_ERROR":"VOLABOT ERROR >"
}}
